### Markdown 支持

博客支持完整的 Markdown 语法：

#### 列表
- 技术文章和教程
- 个人经验和心得
- 当前技术趋势的思考
- 书籍推荐

#### 表格
| 功能 | 描述 | 状态 |
|------|------|------|
| MDX 支持 | 在 Markdown 中使用 React 组件 | 
| 代码高亮 | 使用 Prism.js 实现语法高亮 | 
| 深色模式 | 支持自动和手动切换主题 | 

### 代码块支持

你可以在文章中插入各种语言的代码块：

```python

    return True

if __name__ == "__main__":
    hello_world()
```
